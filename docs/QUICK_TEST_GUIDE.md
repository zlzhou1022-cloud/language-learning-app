## 🐛 常见问题

### Q1: AI没有返回进度？
**检查**:
1. 查看Network标签，确认API返回了 `progress` 字段
2. 查看控制台是否有错误
3. 确认Gemini API Key配置正确

### Q2: 进度一直是0%？
**检查**:
1. 确认AI的回复中包含 `[PROGRESS:XX]` 标记
2. 查看API是否正确解析了进度
3. 查看客户端是否正确更新了状态

### Q3: 进度会减少？
**检查**:
1. 查看 `chat-interface.tsx` 中的 `setAiProgress` 逻辑
2. 应该使用 `Math.max(prev, data.progress)` 确保只增不减
