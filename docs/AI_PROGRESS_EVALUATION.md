# AI驱动的对话质量评估

**更新时间**: 2026-05-02  
**版本**: 2.0.0  
**状态**: ✅ 已实现

---

## 🎯 功能概述

将对话质量评估从客户端规则判断升级为**AI智能评估**。AI在每次回复时，会根据对话的深度、相关性和学习价值，给出一个0-100的进度评分。

---

## 🔄 变更原因

### 之前的问题
- **规则判断不够智能**: 基于关键词匹配，容易被绕过
- **无意义堆叠**: 用户重复输入单词就能增加进度
- **无法判断质量**: 只能统计数量，无法评估对话深度
- **维护成本高**: 需要不断添加关键词和规则

### 现在的优势
- ✅ **AI智能判断**: 理解对话的实际内容和学习价值
- ✅ **防止作弊**: 无意义重复不会增加进度
- ✅ **质量优先**: 关注对话深度而非数量
- ✅ **自动优化**: AI会随着模型升级自动改进

---

## 🧠 AI评分机制

### 评分标准（0-100）

#### 0-20%: 初始阶段
- AI给出了单词的初步介绍
- 用户还未提问或只是打招呼
- **示例**: "你好" → 10%

#### 20-40%: 开始互动
- 用户开始提问，但问题与单词学习无关或关联度很低
- 闲聊、无关话题
- **示例**: "今天天气怎么样？" → 25%

#### 40-60%: 基础学习
- 用户提出了与单词相关的基础问题
- 询问含义、用法等基本信息
- **示例**: "test是什么意思？" → 45%

#### 60-80%: 深入探讨
- 用户深入探讨了2-3个学习要点
- 词源、使用场景、易混淆点、例句、搭配、语气细微差别
- **示例**: "test的词源是什么？" → 65%

#### 80-100%: 优质对话
- 用户深入探讨了4个以上学习要点
- 对话质量很高，足以生成优质学习卡片
- **示例**: "test和exam有什么区别？" → 85%

## 🔒 防作弊机制

### 进度只增不减
```typescript
// 客户端确保进度只能增加或持平
setAiProgress((prev) => Math.max(prev, data.progress));
```

### AI识别无意义重复
```
用户: "test test test test"
AI评分: 46% (只增加1-2%)
原因: 无实质性学习内容
```

### AI识别无关对话
```
用户: "推荐一部电影"
AI评分: 45% (持平)
原因: 与学习单词无关
```

## 🛠️ 技术实现

### 1. Prompt工程

在 `lib/llm/prompts.ts` 中添加了详细的评分指导：

```typescript
export const SYSTEM_ROLE = `你是一个博学、克制、擅长举一反三的语言专家。
...

重要：每次回复时，你需要在回复的最后一行添加一个进度评分标记：
[PROGRESS:XX]

进度评分规则（0-100）：
- 0-20%: 初始介绍阶段
- 20-40%: 用户开始提问，但问题与单词学习无关
- 40-60%: 用户提出了与单词相关的基础问题
- 60-80%: 用户深入探讨了2-3个学习要点
- 80-100%: 用户深入探讨了4个以上学习要点

评分原则：
1. 进度只能增加或持平，不能减少
2. 如果用户的问题与学习单词无关，进度持平
3. 如果用户只是堆叠单词本身而没有实质性学习内容，进度增加很少（+1-2%）
4. 如果用户提出了有深度的问题，进度显著增加（+10-20%）
5. 考虑对话的整体质量，而不仅仅是消息数量
`;
```

### 2. API解析

在 `app/api/chat/route.ts` 中解析AI返回的进度：

```typescript
// 解析进度标记 [PROGRESS:XX]
const progressMatch = response.match(/\[PROGRESS:(\d+)\]/);
let progress = null;
let cleanResponse = response;

if (progressMatch) {
  progress = parseInt(progressMatch[1], 10);
  // 移除进度标记，只返回干净的回复内容
  cleanResponse = response.replace(/\[PROGRESS:\d+\]\s*$/, '').trim();
}

return NextResponse.json({ 
  response: cleanResponse,
  progress: progress 
});
```

### 3. 客户端状态管理

在 `components/learn/chat-interface.tsx` 中管理进度状态：

```typescript
const [aiProgress, setAiProgress] = useState<number>(0);

// 更新AI评估的进度（只增不减）
if (data.progress !== null && data.progress !== undefined) {
  setAiProgress((prev) => Math.max(prev, data.progress));
}
```

### 4. 质量指示器简化

在 `components/learn/conversation-quality-indicator.tsx` 中使用AI进度：

```typescript
// 使用AI评估的进度，而不是客户端计算
const quality = useMemo(() => {
  const progress = aiProgress;
  
  // 根据进度确定等级
  let level: 'insufficient' | 'ready' | 'excellent';
  
  if (progress < 40) {
    level = 'insufficient';
  } else if (progress < 80) {
    level = 'ready';
  } else {
    level = 'excellent';
  }

  return { level, progress };
}, [aiProgress]);
```
