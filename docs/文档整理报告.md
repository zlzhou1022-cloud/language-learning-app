## 🎯 使用指南

### 查找文档

1. **从文档中心开始**: [docs/README.md](./README.md)
2. **按分类查找**:
   - 认证问题 → `docs/auth/`
   - 调试问题 → `docs/debug/`
   - 配置问题 → `docs/setup/`
3. **核心文档**: 直接在根目录查找

### 添加新文档

根据文档类型放入对应文件夹：

```bash
# 认证相关
docs/auth/新文档.md

# 调试相关
docs/debug/新文档.md

# 配置相关
docs/setup/新文档.md

# 核心文档
根目录/新文档.md
```
