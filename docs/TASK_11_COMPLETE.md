## 🎯 任务概述

**问题**: 用户可以在与AI的对话中学习其它单词，导致注意力分散

**解决方案**: AI友好地引导用户回到目标单词的学习上

---

## ✅ 实现内容

### 核心策略：友好引导

当用户偏离主题时，AI会：
1. ✅ 简短回应用户的问题（保持对话自然）
2. ✅ 友好地将话题引回目标单词
3. ✅ 提供相关的学习建议
4. ✅ 保持专业但不生硬

---

[PROGRESS:46]"

**特点**:
- ✅ 认可了用户的话

AI: "很好的问题！exam通常指正式的考试，而test更多指测验或检验。

**特点**:

- ✅ 进度增加（因为是深度学习）

## 🛠️ 技术实现

### 修改的文件: lib/llm/prompts.ts

#### 1. 在 SYSTEM_ROLE 中添加引导原则

```typescript
重要：保持学习焦点
- 用户正在学习一个特定的单词，这是本次对话的核心目标
- 如果用户提到其他单词或偏离主题，友好地引导回到目标单词
- 可以简短回答相关问题，但要自然地将话题引回目标单词
- 不要生硬或说教，保持对话的自然流畅
```

#### 2. 提供引导示例

```typescript
引导示例：
用户："apple这个词怎么用？"（偏离了目标单词test）
AI："apple是个很常用的词呢。不过我们现在主要在学习test这个词，
要不要继续深入了解test的用法？比如test在不同场景下的含义？"
```

```typescript
重要：${word} 是本次对话的核心学习目标。
如果用户偏离主题或询问其他单词，请友好地引导回到 ${word}。
```

#### 4. 更新进度评分规则

```typescript
2. 如果用户的问题与学习单词无关（闲聊、无意义重复、询问其他单词），进度持平
```

## 🎨 引导策略

### 策略1: 简短回应 + 引导
**适用**: 用户询问其他单词
```
"[简短回应其他单词]。不过我们现在主要在学习[目标单词]，
[提供继续学习的建议]？"
```

### 策略2: 建立联系 + 引导
**适用**: 完全无关的话题
```
"[认可用户的话]。说到[话题]，[目标单词]在[相关领域]也有应用，
比如[例子]。回到[目标单词]本身，[提问引导]？"
```

### 策略3: 对比学习 + 聚焦
**适用**: 询问相关单词
```
"很好的问题！[简短对比]。让我们深入了解[目标单词]的[某个方面]..."
```
