  - 添加 GIN 索引以提高查询性能

## 技术细节

### 标签分类逻辑
- 标签基于**单词的语言**，而不是用户学习时使用的语言
- 例如：英语单词的标签会归类到"英语"分类下

### UI/UX 设计
- **列表页**:
  - 语言和标签使用下拉列表（dropdown）而非 tabs
  - 标签显示为小型圆角标签，灰色背景
  - 支持多标签显示

- **详情页**:
  - 标签显示在单词标题下方，带图标
  - 标签管理对话框采用模态窗口
  - 添加标签支持回车键快捷操作
  - 删除标签需要点击 X 按钮

### 数据库性能
- 使用 GIN 索引优化标签数组查询
- 支持高效的标签过滤和搜索
