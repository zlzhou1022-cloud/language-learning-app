# 学习焦点引导功能

**更新时间**: 2026-05-02  
**版本**: 2.0.2  
**状态**: ✅ 已实现

---

## 🎯 功能概述

当用户在对话中偏离主题（询问其他单词或无关话题）时，AI会友好地引导用户回到目标单词的学习上。

---

## 🐛 发现的问题

**现象**: 用户可以在与AI的对话中学习其它单词

**示例**:
```
目标单词: test

用户: "test是什么意思？"
AI: "test是测试的意思..."

用户: "apple这个词怎么用？"
AI: "apple是苹果，可以这样用..." ❌

问题：AI开始教其他单词，偏离了学习目标
```

**影响**:
- 用户注意力分散
- 学习效率降低
- 无法深入学习目标单词
- 进度评估不准确

---

## ✅ 解决方案

### 核心策略：友好引导

AI在检测到用户偏离主题时，会：
1. ✅ 简短回应用户的问题（保持对话自然）
2. ✅ 友好地将话题引回目标单词
3. ✅ 提供相关的学习建议
4. ✅ 保持专业但不生硬

### 引导原则

```typescript
重要：保持学习焦点
- 用户正在学习一个特定的单词，这是本次对话的核心目标
- 如果用户提到其他单词或偏离主题，友好地引导回到目标单词
- 可以简短回答相关问题，但要自然地将话题引回目标单词
- 不要生硬或说教，保持对话的自然流畅
```

## 📊 引导示例

### 示例1: 询问其他单词

**场景**: 用户询问其他单词

```
目标单词: test

用户: "apple这个词怎么用？"
```

**特点**:
- ✅ 简短回应了apple
- ✅ 友好地提醒学习目标
- ✅ 提供了继续学习的方向
- ✅ 进度持平（因为偏离主题）

**场景**: 用户聊天气等无关话题

```
目标单词: test

AI回复:
"是的！说到天气，test这个词在气象学中也有应用，
比如weather test（气象测试）。回到test本身，

[PROGRESS:46]"
```

**特点**:
- ✅ 认可了用户的话
- ✅ 巧妙地将天气和test联系起来
- ✅ 自然地引回学习主题
- ✅ 进度持平

### 示例3: 询问相关但不同的单词

**场景**: 用户询问相关单词

```
目标单词: test

用户: "exam和test有什么区别？"

AI回复:
"很好的问题！exam通常指正式的考试，而test更多指测验或检验。
test可以是小测验，也可以是各种测试。让我们深入了解test的用法...

[PROGRESS:75]"
```

**特点**:
- ✅ 这是有价值的对比问题
- ✅ 回答了区别，但重点在test
- ✅ 继续深入讲解test
- ✅ 进度增加（因为是深度学习）

### 示例4: 连续偏离主题

**场景**: 用户多次偏离

**特点**:
- ✅ 多次友好引导
- ✅ 不会生硬或说教
- ✅ 提供学习建议
- ✅ 进度只在回到主题时增加

## 🛠️ 技术实现

### 1. System Prompt更新

在 `lib/llm/prompts.ts` 的 `SYSTEM_ROLE` 中添加：

```typescript
重要：保持学习焦点
- 用户正在学习一个特定的单词，这是本次对话的核心目标
- 如果用户提到其他单词或偏离主题，友好地引导回到目标单词
- 可以简短回答相关问题，但要自然地将话题引回目标单词
- 不要生硬或说教，保持对话的自然流畅

引导示例：
用户："apple这个词怎么用？"（偏离了目标单词test）
AI："apple是个很常用的词呢。不过我们现在主要在学习test这个词，
要不要继续深入了解test的用法？比如test在不同场景下的含义？"

用户："今天天气真好"（完全无关）
AI："是的！说到天气，test这个词在气象学中也有应用，
比如weather test（气象测试）。回到test本身，你还想了解它的哪些方面呢？"
```

### 2. Initial Prompt更新

在 `INITIAL_WORD_PROMPT` 中强调目标单词：

```typescript
重要：${word} 是本次对话的核心学习目标。
如果用户偏离主题或询问其他单词，请友好地引导回到 ${word}。
```

### 3. 进度评分规则

偏离主题时进度持平：

```typescript
2. 如果用户的问题与学习单词无关（闲聊、无意义重复、询问其他单词），进度持平
```

## 📊 引导策略

### 策略1: 简短回应 + 引导

**适用**: 用户询问其他单词

```
模板：
"[简短回应其他单词]。不过我们现在主要在学习[目标单词]，
[提供继续学习的建议]？"
```

### 策略2: 建立联系 + 引导

**适用**: 完全无关的话题

```
模板：
"[认可用户的话]。说到[话题]，[目标单词]在[相关领域]也有应用，
比如[例子]。回到[目标单词]本身，[提问引导]？"
```

### 策略3: 对比学习 + 聚焦

**适用**: 询问相关单词

```
模板：
"很好的问题！[简短对比]。让我们深入了解[目标单词]的[某个方面]..."
```

### 策略4: 温和提醒 + 建议

**适用**: 多次偏离

```
模板：
"我注意到你对[其他话题]感兴趣。不过让我们先把[目标单词]学透彻，
这样你会有更扎实的学习方法。[提供学习建议]？"
```

## 🎨 引导语气

### ✅ 好的引导

**特点**: 友好、自然、有建设性

```
✅ "apple是个很常用的词呢。不过我们现在主要在学习test..."
✅ "是的！说到天气，test这个词在气象学中也有应用..."
✅ "我注意到你对水果词汇感兴趣。不过让我们先把test学透彻..."
✅ "很好的问题！exam和test确实容易混淆。让我们深入了解test..."
```

### ❌ 不好的引导

**特点**: 生硬、说教、不友好

```
❌ "请不要问其他单词，我们现在学习test"
❌ "你偏离主题了，回到test"
❌ "我只能教test，不能教apple"
❌ "你应该专注于test"
```
